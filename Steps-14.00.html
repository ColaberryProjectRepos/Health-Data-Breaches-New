KPI Cards 14

Create Total Breaches Card

  1. Add a Card to canvas and drag “.Total Breaches” to fields 
  2. In visualizations/visual toggle category label off 
  3. In visualizations/general set height to 153 and width of 202 at position 0 x 137.  
  4. Toggle Title on, type “Total Breaches” in text box.  Set font at Arial black and font size to 12.  Change text color to white and background color to blue 25% darker.  Center the text. 
  5. In effects set the transparency to 25%



Create Breaches per Mth Card


  1. Add a Card to canvas and drag “.Breaches per Month” to fields 
  2. In visualizations/visual toggle category label off
  3. In visualizations/general set height to 153 and width of 202 at position: horizontal 0 and vertical 289.  
  4. Toggle Title on, type “Breaches per Mth” in text box.  Set font at Arial black and font size to 12.  Change text color to white and background color to blue 25% darker.  Center the text.
  5. In effects set the transparency to 25%
  6. Click Breaches per Month measure and format to whole number
