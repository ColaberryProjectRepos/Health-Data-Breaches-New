Create background and title for report 11


  1. Format page change type to custom.  Set height at 900 px and width at 1500 px
  2. Click insert then image
  3. Import databreachphoto1
  4. In format page/canvas background set image to fit and transparency at 60%
  5. Click insert then image and import colaberry photo
  6. Click General in format bar and set size at Height 91 and Width 172, Position at Horizontal 1328 and Vertical 0
  7. Click Insert then text box.  Set Height 90 and Width 1125.  Position Horizontal 202 and Vertical 0
  8. Set font size at 36 and in Bold and Type title “Health Data Breaches 2019 -2021”
