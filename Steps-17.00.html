Arrows for Breaches YoY%

  1. Add New Measure to breach_report table using following DAX measure: 

.DiffsBreach = VAR IconPositive = UNICHAR(9650)                                                                             

VAR IconNegative = UNICHAR(9660)                                                                           

VAR YOYBreachesLY = CALCULATE([.Breaches   YoY%],DATEADD('Date'[Date].[Date],-1,YEAR))                                            VAR Result =  If ([.Breaches YoY%]>YOYBreachesLY,IconPositive,IconNegative)                                             

Return                                                                                                                   

Result

2. Add New Measure to breach_report table using following DAX formula:

.BreachArrowColor =  Var YoYBreachesLY = CALCULATE(breach_report[.Breaches YoY%], DATEADD('Date'[Date].[Date],-1,YEAR))                                                       

Var Result =  If(breach_report[.Breaches YoY%] >  YoYBreachesLY,1,0)               

Return                                                                                                                   

Result

 Arrows for Affected YoY%



1.      Add New Measure to breach_report table using following DAX formula: 
.DiffsInd = 
VAR IconPositive = UNICHAR (9650) 
VAR IconNegative = UNICHAR (9660) 
VAR YOYIndAffectedLY = CALCULATE([.Affected YoY%],DATEADD('Date'[Date].[Date],-1,YEAR)) 
VAR Result = 
    If ( 
    [.Affected YoY%]>YOYIndAffectedLY,IconPositive,IconNegative) 
    RETURN 
    RESULT
2.      Add New Measure to breach_report using following DAX formula: 
.AffectedArrowColor = 
     VarAffectedYoYLY = CALCULATE(breach_report[.Affected YoY%], DATEADD('Date'[Date].                [Date],-1,YEAR)) 
VarResult = 
    If(breach_report[.Affected YoY%] > AffectedYoYLY,1,0) 
    RETURN
    RESULT 
 
3.      Add Card to dashboard and drag DiffsInd into fields 
4.     Format visual: In general/properties- Set height to 46 and width to 53.  Set Position Horizontal to 147 and Vertical to 767 
5.     Format Visual: Set font to 25 and click fx under color for conditional formatting 
6.     Set format style to Rules.  Field should be based on .AffectedArrowColor 
7.     Create rules as follows: 
If value = 0 then RED 
8.     Click New rule and set the next rule: 
If value = 0 then GREEN
