Create Tooltips Cards 18

1.      Click the + sign to open a new page.
2.     Format the visual: Under Page information change name to Tooltip and toggle “allow use as tooltip” on
3.     Click canvas settings.  Change type to tooltip
4.     Copy “Breaches Per Mth, Breaches YoY%, Affected per Breach and Affected YoY% cards and paste onto Tooltips canvas page.
5.     For Affected YoY%, set height to 120 and width to 160.  Set position horizontal to 159 and vertical to 120
6.     For Affected per Breach, set height to 120 and width to 160. Set position horizontal to 0 and vertical to 120
7.     For Breaches Per Mth, set height to 120 and width to 160.  Set position horizontal to 0 and vertical to 0.
8.     For Breaches YoY% set height to 120 and width to 160.  Set positon horizontal to 160 and vertical to 0
