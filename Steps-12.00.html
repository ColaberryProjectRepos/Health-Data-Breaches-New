Create Slicer 12

  1. Add Slicer.  Drag “year” from Date table to fields and toggle off slicer header
  2. Set size of slicer to Height 136 and Width 200.  Set Position of slicer at Horizontal 0 and Vertical 0
  3. Toggle Title on, type “Year” in the text box. Set font at Arial Black 11
  4. Text Color- white and Background Color Blue. Horizontal alignment centered
  5. Click Effects dropdown and toggle background on and change transparency to 25%
