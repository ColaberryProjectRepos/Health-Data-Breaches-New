Visualizations: Map 16

Individuals Affected Average per State



  1. In Map table add a measure.  Use DAX formula:  Ind Affected average per State = AVERAGEX(KEEPFILTERS(VALUES('Map'[State])),CALCULATE(SUM('Map'[Ind Affected]),'Date'[Date]))
  2. Add Shape Map Visual
  3. Drag State to location and Ind Affected average per State to color saturation
  4. Format Visual:  Click dropdown on fill colors and toggle gradient on.  Set minimum color to #f5c4af, medium to #eb895f, maximum to #E66C37.
  5. Set Height to 405 and Width to 654.  Set Position:  Horizontal 202 and Vertical 495
  6. Click insert ribbon, import MapLegend3
  7. Set Height to 61 and Width to 300. Set Position: Horizontal at 506 and Vertical 507
